Política de Privacidade

Última atualização: 24 de Outubro de 2024

O CoraEats valoriza a privacidade de seus clientes e dos usuários finais do sistema. Esta política descreve como tratamos os dados.

1. Dados Coletados

Para a prestação do serviço de licenciamento, coletamos do CONTRATANTE (Lojista): Nome, CPF/CNPJ, E-mail, Telefone e Endereço.

O sistema CoraEats instalado no servidor do lojista coleta dados dos CONSUMIDORES FINAIS (clientes do restaurante) para processamento de pedidos. Esses dados são de responsabilidade do Lojista.

2. Responsabilidade pelos Dados (Controlador vs Operador)

  • O CoraEats (Nós): Atuamos como fornecedores de tecnologia. Temos acesso aos dados do Lojista para fins de faturamento e suporte.
  • O Lojista (Você): É o Controlador dos dados dos seus clientes finais. Você é responsável por garantir que seu uso do sistema esteja em conformidade com a LGPD perante seus clientes.

3. Hospedagem de Dados

O sistema é hospedado em servidores gerenciados pela CONTRATADA (utilizando infraestrutura Hostgator). Embora a CONTRATADA gerencie o servidor para fins de manutenção e segurança, todos os dados de pedidos, clientes e produtos pertencem exclusivamente ao Lojista.

4. Compartilhamento

Não vendemos, alugamos ou compartilhamos dados pessoais com terceiros para fins de marketing.
